# Interactive Tutorial

Prisma Calendar ships with a short, hands-on tour that runs the first time you open it. Instead of reading docs to get started, you learn by doing — the tour creates a real sample event on your calendar and walks you through the things you'll do every day.

The whole thing takes under a minute, and you can replay it whenever you like.

## What the tour shows you

The tour highlights one piece of the calendar at a time, with a short tooltip and an arrow pointing at exactly what it's describing:

1. **Welcome** — a quick hello and what to expect.
2. **Your first event** — the tour creates a sample event called **"Your first event"** on today, then points at it to explain that every event is just a note with a date in its frontmatter.
3. **Move it around** — drag the event to reschedule it, or drag an edge to change its duration. You can interact with the calendar while the tour is open, so go ahead and try it.
4. **Open the editor** — double-click an event to edit its title, time, categories, recurrence, and more.
5. **Create new events** — where the Create button lives (you can also click any empty slot on the grid).
6. **Switch how you see things** — jump between year, month, week, and day, and discover the timeline, heatmap, and dashboard tabs.
7. **You're all set** — where to go next.

Use **Next** / **Back** to move through the steps, **Skip** to leave at any time, or press **Escape** to close it.

## When it runs

The tour starts automatically the first time you open Prisma Calendar, right after you finish the welcome setup. If you close Obsidian partway through without finishing or skipping it, it picks back up the next time you open the calendar — so you won't miss it just because you stepped away. Once you've finished or skipped it, it won't pop up again on its own.

## Replaying the tour

You can run the tour again at any time:

- **Settings → General → Interactive tutorial** — click **Replay tutorial**.
- **Command palette** — run **Prisma Calendar: Start onboarding tutorial**.

Replaying reuses the sample event if it's already on today's calendar, so you won't pile up duplicates.

## The sample event

"Your first event" is a normal event note — nothing special. Keep it, edit it, or delete it like any other event once you're comfortable. If it isn't on today's view when you replay the tour, a fresh one is created so there's always something to point at.
