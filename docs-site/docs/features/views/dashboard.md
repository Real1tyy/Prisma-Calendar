# Dashboard (Pro)

The Dashboard tab provides a comprehensive overview of all your events across three sections — Recurring Events, Categories, and By Name. Each section includes a pie chart distribution and a full sortable table with colored rows, giving you detailed statistics at a glance.

:::info Pro Feature
The Dashboard tab requires a Pro license. Free users will see a prompt to upgrade when switching to this tab.
:::

## Sections

### Recurring Events

Shows all recurring events (enabled and disabled) with their recurrence type, instance count, category, and status. The pie chart shows the distribution of recurrence types (Daily, Weekly, Monthly, etc.). Click any row to open the Event Series Modal for that recurring event.

### Categories

Lists all categories with total event count, timed vs all-day breakdown, and percentages. Rows are colored with their assigned category color. The pie chart shows category distribution by event count. Click any row to open the Event Series Modal filtered to that category.

### By Name

Groups events by their cleaned name (name-based series). Shows the event count per name group. Only visible when name series tracking is enabled in settings. Click any row to open the Event Series Modal for that name group.

## Features

- **Search/filter**: Each section has a filter input to quickly find items by name.
- **Sortable columns**: Click any column header to sort ascending or descending.
- **Colored rows**: Category and recurring event rows are tinted with their resolved color for visual identification.
- **Pie charts**: Each section includes a distribution pie chart showing the breakdown at a glance.
- **Collapsible sections**: Click any section header to collapse or expand it.
- **Pagination**: Tables with more than 20 items are paginated with prev/next controls.
- **Reactive updates**: The dashboard automatically refreshes when events, categories, or recurring events change.
- **Responsive layout**: On narrow screens, charts and tables stack vertically.
