# Prisma Calendar — Claude Instructions

## Documentation

When writing or updating documentation (changelog, feature docs, configuration docs), follow the guidelines in [docs/documentation.md](./docs/documentation.md).

## Changelog

Every feature or fix must include a changelog entry and corresponding documentation updates.

- Write the changelog entry under the **latest version** already present in `docs-site/docs/changelog.md`, updating the date if needed.
- Only create a new version section if explicitly told to target a different version.
